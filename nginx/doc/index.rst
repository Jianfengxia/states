Nginx
=====

Introduction
------------

Nginx (pronounced "engine-x") is an open source reverse proxy server for :ref:`glossary-HTTP`,
:ref:`glossary-HTTPS`, :ref:`glossary-smtp`, :ref:`glossary-pop3` and :ref:`glossary-imap` protocols, as well as a load balancer, :ref:`glossary-HTTP` cache,
and a web server (origin server).
