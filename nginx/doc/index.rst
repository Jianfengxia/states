Nginx
=====

.. Copied from http://en.wikipedia.org/wiki/Nginx on 2014-12-13

Nginx (pronounced "engine-x") is an open source reverse proxy server for :ref:`glossary-HTTP`,
:ref:`glossary-HTTPS`, SMTP, POP3, and IMAP protocols, as well as a load balancer, :ref:`glossary-HTTP` cache,
and a web server (origin server).
