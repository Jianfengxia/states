Introduction
------------

:doc:`index` is a `Unix shell <http://en.wikipedia.org/wiki/Unix_shell>`_
written by Brian Fox for the GNU Project as a free
software replacement for the Bourne shell (sh).

:doc:`index` is a command processor that typically runs in a text window,
where the user types commands that cause actions.
:doc:`index` can also read commands from a file, called a script.
