DjangoPypi 2
============

Introduction
------------

DjangoPyPI is a :doc:`/django/doc/index` application that provides a
re-implementation of the `Python Package Index
<https://pypi.python.org/pypi>`_. Using `Twitter Bootstrap
<http://getbootstrap.com>`_ for UI, forked from the original DjangoPyPi project,
DjangoPyPi2 provides an easy to use and manage interface.

.. https://djangopypi2.readthedocs.org/en/latest
