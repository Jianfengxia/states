GnuPG
=====

GnuPG_ is a complete and free implementation of the OpenPGP standard as defined
by `RFC4880 <http://www.ietf.org/rfc/rfc4880.txt>`_ (also known as PGP). GnuPG_
allows to encrypt and sign your data and communication, features a versatile key
management system as well as access modules for all kinds of public key
directories.
