Introduction
------------

Ehcache is a widely used open source Java distributed cache for general purpose
caching, Java EE and light-weight containers[clarify]. It features memory and
disk stores[clarify], replicate by copy and invalidate[clarify], listeners, cache
loaders, cache extensions[clarify], cache exception handlers, a gzip caching
servlet filter, RESTful and SOAP APIs.[2] Ehcache is available under an Apache
open source license and is actively supported.
