Pillar
======

Optional
--------

terracotta:environment
~~~~~~~~~~~~~~~~~~~~~~

Default: ``production``.

terracotta:java_ms
~~~~~~~~~~~~~~~~~~

Set initial Java heap size (-Xms).

Format: <size>[g|G|m|M|k|K].

Default: ``512m``.

terracotta:java_mx
~~~~~~~~~~~~~~~~~~~~~~

Set maximum Java heap size (-Xmx).

Format: <size>[g|G|m|M|k|K].

Default: ``512m``.
