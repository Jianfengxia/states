Pillar
======

Mandatory
---------

Example::

  android:
    buildtools_versions:
      - 21.1.2
      - 22.0.1
    sdk_api_versions:
      - 22
      - 21

android:buildtools_versions
---------------------------

List of buildtools versions to install.

android:sdk_api_versions
------------------------

List of SDK API versions to install.
