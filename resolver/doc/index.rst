Resolver
========

Introduction
------------

The resolver is a set of routines in the C library that provide access
to the Internet Domain Name System (:ref:`glossary-DNS`).
The resolver configuration file contains information that is read by the
resolver routines the first time they are invoked by a process. This formula
for managing resolver configuration files.

..  http://manpages.ubuntu.com/manpages/trusty/en/man5/resolver.5.html
