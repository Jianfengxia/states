Pillars
=======

Mandatory
---------

Example::

  mail:
    mailname: somehost.fqdn.com

mail:mailname
~~~~~~~~~~~~~

Fully qualified domain (if possible) of the mail server hostname.

Optional
--------

Example::

  mail:
    maxproc: 2

mail:maxproc
~~~~~~~~~~~~

Maximum number of process.

Default: ``2`` by default of that pillar key.
