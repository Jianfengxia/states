strongSwan
==========

:doc:`index` is a complete IPsec implementation for Linux 2.6 and 3.x kernels.
The focus of the project is on strong authentication mechanisms using X.509
public key certificates and optional secure storage of private keys on
smartcards through a standardized PKCS#11 interface.

.. https://en.wikipedia.org/wiki/StrongSwan - 2015-07-09

.. note::

   Do not install this formula on the same machine with
   :doc:`/openswan/doc/index`.
