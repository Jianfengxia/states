Graylog2 Web Server
===================

:doc:`/graylog2/doc/index` provides a powerful web interface:

* Any analysis or search result can be added as a widget to dashboards.

* Build pages that show you all important data and trends with one click.

* Constantly updated and dynamically cached to save resources.

* Alerts allow you to be notified in case of different highly configurable
  conditions or events.
