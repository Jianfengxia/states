S3cmd
=====

Introduction
------------

:doc:`index` is a free command line tool and client for uploading, retrieving
and managing data in Amazon :ref:`glossary-s3` and other cloud storage service
providers that use the :ref:`glossary-s3` protocol, such as Google Cloud
Storage or DreamHost DreamObjects. It is best suited for power users who are
familiar with command line programs. It is also ideal for batch scripts and
automated backup to :ref:`glossary-s3`, triggered from cron, etc.
