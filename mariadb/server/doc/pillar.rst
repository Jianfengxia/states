.. _pillar-mysql-ssl:

mysql:ssl
~~~~~~~~~

The name of the :doc:`/ssl/doc/index` certificate and key files to use for
establishing a secure connection.

Default: Not use (``False``).

.. _pillar-mysql-bind:

mysql:bind
~~~~~~~~~~

What IP does :doc:`/mariadb/server/doc/index` will bind to.

Default: ``127.0.0.1``.

.. _pillar-mysql-password:

mysql:password
~~~~~~~~~~~~~~

Root password used for installation process, change this after.
:doc:`/mariadb/doc/index` installed does not make sense.

Default: ``None`` - randomly generated by ``salt['password.pillar']``.
