PPP
===

Introduction
------------

In computer networking, Point-to-Point Protocol (PPP) is a data link protocol
used to establish a direct connection between two nodes. It can provide
connection authentication, transmission encryption (using ECP, RFC 1968), and
compression.

.. https://en.wikipedia.org/wiki/Point-to-Point_Protocol - 2015-04-10
