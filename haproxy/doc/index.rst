HAProxy
=======

Introduction
------------

HAProxy is a free, open source high availability solution, providing load
balancing and proxying for :ref:`glossary-TCP` and :ref:`glossary-HTTP`-based
applications by spreading requests across multiple servers. It is written in
:ref:`glossary-C` and has a reputation for being fast and efficient (in terms
of processor and memory usage).
