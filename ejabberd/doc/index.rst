Ejabberd
========

Ejabberd is a Jabber/XMPP instant messaging server

Security Issue
--------------

Since we are using version 2.1.10, there is no way to disable SSLv3 without
re-compiling the source code. This is a dirty hack, so we will leave it like
that until there is a build for Ubuntu that fix it.
