Kernel
======

Introduction
------------

In computing, the :doc:`index` is a computer program that manages I/O
(input/output) requests from software, and translates them into data processing
instructions for the central processing unit and other electronic components of
a computer. The kernel is a fundamental part of a modern computer's operating
system.

.. https://en.wikipedia.org/wiki/Kernel_(operating_system) - 2015-07-02

Linux kernel
~~~~~~~~~~~~

The Linux kernel is a Unix-like computer operating system kernel. The Linux
kernel is a widely used operating system kernel world-wide; the Linux operating
system is based on it and deployed on both traditional computer systems,
usually in the form of Linux distributions,[9] and on embedded devices such as
routers. The Android operating system for tablet computers and smartphones is
also based atop the Linux kernel.

.. https://en.wikipedia.org/wiki/Linux_kernel - 2015-07-02
