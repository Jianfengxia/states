Pillar
======

Optional
--------

Example::

  nodejs:
    version: 4.1.1-1

nodejs:version
~~~~~~~~~~~~~~

Which version of :doc:`index` to install.

Possive values:

* Precise

  * ``0.10.32`` (default)
  * ``4.1.1-1``

* Trusty

  * ``0.10.25`` (default)
  * ``4.1.1-1``

Default: depends on Ubuntu release (``None``).

.. warning::

   Switching between :doc:`index` versions is not supported.
