      username: cleaner
      token: JENKINS_APITOKEN_FOR_CLEANER

jenkins:job_cleaner
~~~~~~~~~~~~~~~~~~~

:doc:`index` user used for deleting old disabled jobs.

.. note::

Default: does not use ``False``.

jenkins:job_cleaner:days_to_del
